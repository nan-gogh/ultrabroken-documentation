# Void Holding

## Overview
Allows you to hold materials "whenever you want" as long as you are at a low enough Y coordinate

## How To
```
Clip below roughly Y -4000 and hold any stack of items
```

## Metadata
**Working versions:** 1.0.0
1.1.0/1.1.1
**Credit:** Lightos_ - June 10, 2023
**Source:** Discord
**Effects/Extensions:** Can cause a weird hold smuggeling state that usually softlocks the game after a few seconds, however you can still do actions (e.g. warping), as long as you do it before the softlock
**Examples:** https://discord.com/channels/1086729144307564648/1105598687167664239/1117479024063684660
