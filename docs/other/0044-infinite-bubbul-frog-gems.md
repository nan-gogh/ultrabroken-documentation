# Infinite Bubbul Frog Gems

## Overview
Allows you to get infinite bubbul gems

## How To
```
Only works in some caves!
1. Shoot the bubbul frog
2. Ascend during it's death
3. Cancel the ascend
4. Shoot the bubbul frog
5. Repeat
```
