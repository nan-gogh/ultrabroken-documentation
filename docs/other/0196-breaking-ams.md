# Breaking AMS

## Overview
By using certain setups, it is possible to ignore the no durability use.

## How To
```
Method 1:
1. Activate arrow smuggle
2. activate sidons ability
3. mount mineru
4. jump off and enter bullet time
5. jump slash with the ams

Method 2:
1. Activate arrow smuggle
2. hold out the bow
3. side hop
4. press Y
```
