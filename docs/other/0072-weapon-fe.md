# Weapon FE

## Overview
Allows for FE to weapons

## How To
```
1. Fuse entangle weapon to shield 
2. Zuggle a different weapon
3. Pick up fe weapon
4. Do normal fuse entanglement setup but begin fusing to weapon instead 
5. Swap off shield with weapon
```
