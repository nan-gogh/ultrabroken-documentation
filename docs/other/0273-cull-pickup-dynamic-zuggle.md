# Cull Pickup Dynamic Zuggle

## Overview
Picking up a thrown boomerang on the same frame that it culls can lead to dynamic zuggle

## How To
```
1. Fuse entangle a boomerang to a shield
2. Drop the shield the boomerang is fuse entangled to in a culling area
3. Throw the boomerang standing near the edge of the culling area
4. Walk out of the culling area as the boomerang approaches Link, catch it on the same frame that it culls and immediately pause
5. Drop the boomerang, equip a different weapon, and then unequip that weapon before unpausing
6. Pickup the dropped boomerang and switch weapons
7. Drop the equipped weapon to zuggle
8. Detangle the dynamic zuggle from the shield
```

## Metadata

**Effects/Extensions:** Works with any weapon that can be thrown instead of only boomerangs if combined with fail drop smuggle
**Notes:** Possible to perform on 1.1.2+ with force swap on step 6. 1.2.0+ requires manual unculling of the dynamic smuggle after force swap
