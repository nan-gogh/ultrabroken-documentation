# Fuse Entanglement

## Introduction

Fuse Entanglement describes interactions where objects and game state become linked in unexpected ways, producing persistent or shared behavior between items and entities. This section collects notes and reference material about entanglement-style behaviors.

## Quick Navigation

- **[Fuse Entanglement](0051-fuse-entanglement/)** Basic setups for Fuse Entanglement

## What is Fuse Entanglement?

Fuse Entanglement refers to situations where two or more objects become bound so that actions on one affect the other. These behaviors are often central to advanced Ultrabroken setups and can enable or disable specific effects depending on the entangled pair.
