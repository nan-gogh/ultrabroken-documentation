# Cull Cold Fuse

## How To
```
1. place object in tarrey town culling area
2. move outside culling area while having the object in ultrahand focus
3. select fuse and pause
5. watch a memory
6. unpause and immediately fuse the object (fuse stored)
7. stand against a hover stone as wall (do not move further away from the culling area!)
8. watch a memory
9. drop your equipment, swap to another, drop that as well and unpause
if done correctly, the object will teleport to Link

steps 8-9 are not required on 1.0.0
```
