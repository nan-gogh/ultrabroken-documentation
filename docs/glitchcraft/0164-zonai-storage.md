# Zonai Storage

## Overview
Allows Zonai devices to be stored and taken into shrines

## How To
```
1. Select any Zonai device 
2. Press A and B at the same time
You now have stored the zonai device!
If you enter a shrine you may chose 1 to 10 zonai devices to spawn
```

## Metadata
**Working versions:** 1.0.0
**Credit:** bebu0815 - August 13, 2023
**Source:** Discord
**Effects/Extensions:** Activating zonai storage, dying and opening the inventory may crash the game
