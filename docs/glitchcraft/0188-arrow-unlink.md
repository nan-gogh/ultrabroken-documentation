# Arrow Unlink

## Overview
makes arrow float away allowing you to spawn fused stuff in other loactions

## How To
```
1.Zuggle until your bow drops 
2.Fuse the bow on the ground to a sheild
3.let the arrow float away then drop the sheild
```

## Metadata
**Working versions:** 1.0.0
**Credit:** R4000 - October 26, 2023
**Source:** Discord
**See also:** Zuggle
Zuggle Overload
**Effects/Extensions:** Can cause some form of warping with wings, springs and big wheels
**Examples:** https://discord.com/channels/1086729144307564648/1105598687167664239/1167228288834486362
**Notes:** it is very inconsistent and is still being tested
