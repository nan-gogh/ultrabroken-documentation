# Like-Like FSFE

## Overview
A method of Fuse Storage Fuse Entanglement.

## How To
```
1. Drop target item in front of a like-like
2. Fuse the target item to a weapon or shield on the same frame that a like like eats it
3. Stun the like like and pause on the frame the fuse stored item unculls
4. Swap to a different item of the same type as the one you fused with (unequipping and reequipping the same item also works)
```
